Experience the consistent, beautiful popups across StreetEats Connect

Modal Popups

These are the main popup modals that appear for confirmations, alerts, and information display.

Confirmation Dialogs

Interactive confirmation dialogs for user actions that require approval.

Toast Notifications

Non-intrusive toast notifications that appear in the corner for quick feedback.

Loading Modal

Loading modals for operations that take time to complete.

How to Use

The themed popup system is automatically available on all pages. Here are some examples:

// Show an alert
themedPopup.showAlert('Your message here', 'Title', 'info');
// Show a confirmation
const confirmed = await themedPopup.showConfirm('Are you sure?', 'Confirm Action');
if (confirmed) { /* do something */ }
// Show toast notifications
themedPopup.success('Operation completed successfully!');
themedPopup.error('Something went wrong!');
themedPopup.warning('Please check your input.');
themedPopup.info('Here\'s some information.');
// Show loading modal
const loading = themedPopup.showLoading('Processing...');
// Close it later: themedPopup.closeModal();
